# FlagPost: Puzzle MVP Scope

## Overview
- Product Name: FlagPost: Puzzle
- Package Name: app.flagpost.puzzle
- Concept: A casual sliding puzzle game inspired by the classic 15-puzzle. The player rebuilds a national flag by sliding tiles and then discovers the country.

## MVP Scope
- Core sliding puzzle mechanics.
- HomeScreen with navigation to Game, Collection, and Settings.
- GameScreen with puzzle interaction, timer, and move counter.
- CollectionScreen placeholder for solved flags.
- SettingsScreen placeholder for sound, vibration, and difficulty.

## Explicitly Excluded
- Online and backend features.
- Ads and in-app purchases in MVP.
- Leaderboards and multiplayer features.

## Planned Later Features
- Dynamic flag progression and level expansion.
- Persistent local collection progress.
- Sound effects and haptics.
- Multiple difficulty presets.
- Post-solve country trivia expansion.
